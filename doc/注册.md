
    
**简要描述：** 

- 用户注册接口

**请求URL：** 
- ` /register `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|password |是  |string | 密码    |
|username     |是  |string | 用户名 |
|phoneNumber| 是 | string| 电话号码|
|email|是|string|邮箱|
|verCode|是|int|验证码|

 **返回示例**

``` 
{
	"mes": "",
	"data": "注册成功",
}
```
