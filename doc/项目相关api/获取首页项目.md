
    
**简要描述：** 

- 获取首页项目

**请求URL：** 
- ` /items `
  
**请求方式：**
- GET

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|page |否  |int | 页数（从0开始）    |
|size |否  |int | 每页大小    |
|location|否|string|位置（省， 例：北京）|

 **返回示例**

``` 
{
    "data": [
        {
            "currentCost": 200,
            "detail": "项目细节",
            "id": 1,
            "imageUrl": "44444",
            "name": "test",
            "originalCost": 100,
            "type": "\u81ea\u52a8",
            "location": "北京"
        }
    ],
    "mes": ""
}
```
