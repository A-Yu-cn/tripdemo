
    
**简要描述：** 

- 用户登录接口

**请求URL：** 
- ` /users/auth `
  
**请求方式：**
- POST 

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|password |是  |string | 密码    |
|email |是  |string | 邮箱    |


 **返回示例**

``` 
{
	"mes": "",
	"data": {
		"id": 4,
		"token": "27ed2a5a45fefd9672d63940ddeb95e"
	},
}
```

**返回值说明**
data中的内容是用户登录时生成的token，有效期一个月
