
    
**简要描述：** 

- 发送验证码接口

**请求URL：** 
- ` /users/{id} `
  
**请求方式：**
- GET

**参数：** 

|参数名|必选|类型|说明|
|:----    |:---|:----- |-----   |
|id |是  |int | 用户ID   |

    },
